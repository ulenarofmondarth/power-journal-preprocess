.. _prerequisites:
Prerequisites
=============

.. rubric:: nodeJS

This project is tested against :program:`nodejs` version 22.14.0.

`Install nodejs <https://nodejs.org/en>`_

.. rubric:: pnpm

The package manager of choice.

`Install pnpm <https://pnpm.io/installation>`_

(See :ref:`replacing pnpm` if you object to efficiency.)

.. rubric:: docker

(And :program:`docker compose`---commonly bundled when you install Docker.)

`Install docker <https://www.docker.com/>`_

* :program:`docker` version v28+
* :program:`docker compose` version v2.33+

.. rubric:: git

Pretty much any modern :program:`git`, at the time of writing I have v2.39.5 installed.

`Install git <https://git-scm.com/downloads>`_
