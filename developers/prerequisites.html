Prerequisites

nodeJS

This project is tested against nodejs version 22.14.0.

Install nodejs

pnpm

The package manager of choice.

Install pnpm

(See Can I use npm instead of pnpm? if you object to efficiency.)

docker

(And docker compose—commonly bundled when you install Docker.)

Install docker

  • docker version v28+

  • docker compose version v2.33+

git

Pretty much any modern git, at the time of writing I have v2.39.5 installed.

Install git
